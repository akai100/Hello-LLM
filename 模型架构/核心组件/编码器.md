
**核心定位**：信息提取器、特征编码器，核心是 **“无损/低损的信息压缩和语义表征学习”**。

+ 输入：原始未加工的输入数据，可为序列型（文本、语音、时序信号）、空间型（图像、视频帧）、多模态融合数据；

+ 输出：结构化的特征表示（可称为上下文向量/隐藏状态/特征图/嵌入向量），该表示需完整、准确捕捉输入的核心语义 / 特征信息，且具备泛化性；

**核心要求**

1. 特征提取的**完整性**：不丢失输入的关键信息

2. 语义的**一致性**：特征表示与输入的语义 / 特征一一对应

3. 维度的**合理性**：将高维原始输入映射为低维 / 固定维 / 可变维的特征，便于解码器处理；

**核心特点**

无生成约束，仅关注 “如何理解输入”，通常采用**双向信息交互**（如双向自注意力、双向 RNN），充分捕捉输入的全局 / 局部特征。
